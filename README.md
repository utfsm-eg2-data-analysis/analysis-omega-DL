# analysis-omega-DL
Codes developed for the analysis of the hadronization of the omega(782) meson via Deep Learning methods, using data and simulations from the EG2 experiment of the CLAS Collaboration.

## Requirements
- Python 3.8.8
- upROOT 4.0.11
- torch 1.9.0
